# Randle

**Randle** is a [Slack](https://slack.com) bot app to provide the features of a **text-based virtual tabletop** by supporting dice, cards, and polls for gaming communities. It is open source under the MIT license by Arcanigma Foundation and free to serve for your own community.

The app is written in strict [TypeScript](https://www.typescriptlang.org/) for [Node.js](https://nodejs.org/en/) using Slack's [Bolt](https://slack.dev/bolt-js/) API. It is intended to be served by a [Heroku](https://www.heroku.com) web dyno, but it can be adapted for other hosting solutions.
